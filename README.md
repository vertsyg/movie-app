# Movie app

## Тестовое задание vk

## Запуск проекта

### 1. Введите в терминале:

>git clone https://github.com/vertsyg/movie-app.git

>npm install

>npm run dev

### 2. Проект также можно открыть на github-pages по ссылке: https://vertsyg.github.io/movie-app/

## Технологии:
- React `v18`
- Redux
- TypeScript
- React-router-dom `v6`
- MUI
- [Кинопоиск API](https://kinopoisk.dev/)
